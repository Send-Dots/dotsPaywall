Paywall your blog posts or articles behind micro-transactions

== Description ==

Add a paywall to your articles or blog posts that can be unlocked by users with [Dots](https://senddots.com) accounts.

Surround the paywalled contents with the `[paywall]` short code. Example:

```
Lorem ipsum dolor sit amet, consectetur...

[paywall price="10"]
Paywalled content .... Lorem ipum dolor sit amet, consectetur...
[/paywall]
```

=== Installation === 

Setup your Dots username in the Dots Paywall settings.
